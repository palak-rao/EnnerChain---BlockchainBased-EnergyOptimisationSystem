CProject Name: Blockchain-Based Smart Energy System

Description:
This project is a smart energy management system that uses blockchain technology.
It helps in distributing energy efficiently and securely.

Problem:
Energy is often wasted and there is no transparency in distribution.

Solution:
This system uses blockchain to store energy transactions securely.
It also uses Dijkstra algorithm to find the best path for energy transfer.

Features :
- Blockchain simulation
- Energy routing using shortest path
- Mining (Proof of Work)
- Dashboard interface

How to Run:
1. Download the project
2. Open  EnnerChain -Blockchain Based.html in your browser
3. Enter energy demand
4. Click submit and mine block

Technologies Used:
- HTML, CSS, JavaScript
- Blockchain (simulation)
- Dijkstra Algorithm

Future Scope:
- Use real blockchain (Polygon)
- Add AI for prediction
